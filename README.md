# This is Disease Predictor Django App

## How to setup:-
1. Make sure you have python > 3.8
2. Install virtualenv ```python -m pip install virtualenv```
3. Create the env folder ```virtualenv env```
4. In windows ```env/Scripts/Activate.Ps1``` make sure you have allowed the powershell to run the scripts.
5. Install all the packages required to run that app ```pip install -r requirements.txt```
6. If all went good then start the server ```python manage.py runserver```
